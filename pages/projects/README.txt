For you project to show up and be linked to a sub team it needs two tags

- The tag project defines its a project such that it will be listed on the project page
	Project is for ongoing projects.
	If you change it to project_old its a project that is finisched.

- The tag grp_xxx defines to which subgroup its linked
	xxx can be one of the following
		- meta (metabolic imaging)
		- plat (platform and methods)
		- pre (preclinical)
		- rec (recon)
		- rf (RF and safety)
		- vasc (vasular brain)
		- coil (coil lab)
		- soft (software)

Example

tags:
	project
	grp_meta
